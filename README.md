### Установка
# Step 1
Скронируйте репозиторий на локальную машину:
```
git clone git@github.com:Studio-Yandex-Practicum/RandomCoffeeBot.git
```
# Step 2
Установите poetry
```
curl -sSL https://install.python-poetry.org | python3 -
```

**Note**
Не используйте pip install poetry
